About the Patient Record System

This Patient Record System is built to simplify and secure the way doctors manage patient records. Features include:

  • ➕ Add patient records with ease
  • 👁️ View and search patients quickly
  • ✏️ Edit or update medical information
  • 🗑️ Delete records securely

Built using Node.js, Express, MongoDB, EJS, and Bootstrap. Secure, scalable, and ready to use.
